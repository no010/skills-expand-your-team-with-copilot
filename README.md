## 数独游戏 (Sudoku Game)

这个仓库包含一个用Python实现的完整数独游戏。

### 功能特点

- 🎯 **数独谜题生成**: 支持简单、中等、困难三种难度
- 🎮 **交互式游戏**: 命令行界面，支持放置数字、移除数字等操作
- 💡 **提示系统**: 可以获取正确答案提示
- ✅ **输入验证**: 自动检查移动是否符合数独规则
- 🔍 **完整性检查**: 自动检测游戏是否完成

### 使用方法

#### 开始游戏
```bash
python3 sudoku.py
```

#### 查看演示
```bash
python3 demo.py
```

### 游戏规则

数独是一个9×9的网格游戏，规则如下：
1. 每行必须包含数字1-9，不能重复
2. 每列必须包含数字1-9，不能重复  
3. 每个3×3的小方格必须包含数字1-9，不能重复

### 游戏操作

- **放置数字**: 输入 `行 列 数字` (例如: `1 2 5`)
- **移除数字**: 输入 `r 行 列` (例如: `r 1 2`)
- **获取提示**: 输入 `hint`
- **退出游戏**: 输入 `quit`

### 系统要求

- Python 3.6 或更高版本
- 无需额外依赖库 (仅使用Python标准库)
